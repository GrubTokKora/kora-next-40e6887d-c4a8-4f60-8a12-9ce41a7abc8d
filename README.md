# Kora Next.js Cloudflare Boilerplate

Minimal Next.js App Router template for Kora-generated business websites.

- Static export via `next.config.mjs` (`output: "export"`)
- SEO-ready App Router structure
- Cloudflare Pages deployment workflow
- Plain CSS starter styles

GitHub should mark this repository as a template so `kora-agent` can create one per-business repo.
